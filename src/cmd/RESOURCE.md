# Cmd 模块资源

## 1. 模块定位

- 路径：`src/cmd`
- 主文件：`src/cmd/main.mbt`
- 职责：仓库主入口，串联 demo 与测试执行。

## 2. 对象说明

- 本模块无独立实体、VO、DTO。
- 本模块主要承载执行流程。

## 3. 执行顺序

- `main`
  1. 输出标题
  2. 执行 `run_demos()`
  3. 执行 `run_tests()`

## 4. Demo 调度

- `run_demos()` 依次执行：
  - 基础压缩
  - 文件分析
  - 数据完整性分析
  - 格式检测
  - 压缩级别比较
  - 压缩方法比较
  - 高级压缩级别演示
  - Huffman 演示
  - 流式压缩演示
  - FSE 调试占位

## 5. Test 调度

- `run_tests()` 依次执行：
  - 保存压缩文件供官方工具验证
  - 编码测试
  - 基础测试
  - 兼容性测试
  - 字典测试
  - 非兼容缺陷回归与交叉兼容测试

## 6. 使用说明

- 推荐命令：`moon run src/cmd`
- 用途：一键跑通当前仓库主要演示与测试路径。
